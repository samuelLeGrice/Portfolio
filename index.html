GAZETTEER PROJECT

The project...

The Gazetteer is an engaging and informative application that offers easy-to-use features, allowing the user to find the following for either their current location, if shared, or a country of their choosing:

  • cities & airports;
  • country information;
  • country-relevant news & Wikipedia articles;
  • weather forecast;
  • public holidays; and
  • currency conversions.

Stack Used...

The front-end is developed using jQuery and JavaScript, with PHP handling the back-end processes. The PHP back-end queries multiple APIs to fetch data, which is then parsed and displayed on the front-end using a combination of Bootstrap, HTML, and CSS. This setup ensures a seamless integration of dynamic content and responsive design.

The Gazetteer application was developed using Visual Studio Code.

COMPANY DIRECTORY PROJECT

The project...

The Company Directory application makes it easy for a company to manage personnel information. To be able to do this I've added the ability to:

  • Search through the company's personnel, departments and locations separately.
  • Add, edit and delete personnel.
  • Search by name across any of the categories.
  • Add specific filters for each category.

There are also measures put in place to ensure that vital database information cannot be deleted.

Stack used...

The front-end has been created using HTML, CSS, Javascript, bootstrap and JQuery allowing for a dynamic and responsive front-end with a crisp and professional appearance. The back-end makes use of PHP ands SQL to perform CRUD operations on a MySQL database.

The Company Directory application was developed using Visual Studio Code.
